# image-divider

usage:
```shell
Usage of image-divider.exe:
  -clips int
        How many clips will the image be divided (default -1)
  -f string
        The format of output image files (default "png")
  -i string
        Image file
  -o string
        Output Directory
  -p string
        How many clips will the image be divided (default -1)
  -f string
        The format of output image files (default "png")
  -format string
        The format of output image files (default "png")
  -i string
        Image file
  -input string
        Image file
  -o string
        Output Directory
  -output string
        Output Directory
  -p string
        Prefix of the output files
  -prefix string
        Prefix of the output files

```
